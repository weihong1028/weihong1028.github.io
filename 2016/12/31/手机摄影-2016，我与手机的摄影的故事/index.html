Instagram 和 vsco 带来的改变

　　对于一个喜爱摄影的人，毕业之后最大的挑战可能是如何坚持摄影了吧。至少于我而言是这样的。在大学你有大量的时间去玩摄影，但离开学校后有过一段时间的思想挣扎，单反几乎不用，不知道拍些什么，不知道自己想要的摄影美是什么。那些日子，摄影就像件等待被抛弃的旧衣裳。
　　值得庆幸的是，Instagram 和 vsco 的出现改变了这一切。

器材

　　离开学校后在日常生活中用单反的机率很小，所幸智能手机的发展使得手机摄影越来越贴近于我的生活，从早期陪着我尝试手机摄影的是红米手机，到 2016 年 4 月我有了第一部 iphone 手机。

观念

　　以前在大学用单反拍得最多的是人文纪实，风景，但离开学校后，你不能再享受花上一个下午的时间去构思一幅作品的待遇，也不能走街串巷寻觅下一个人文情怀的爆发点。
　　这时，Instagram 给我带来了转折点。Instagram 是一个高质量的图片社区，看到一些牛人们的作品，往往会给我带来一种视觉上的震撼。真的很让人享受，美到心坎里去。*所以这个时期的我也慢慢专注于拍自己觉得美的东西 ( 比如：建筑、花儿、小风小景，静物，简称小清新系列 ) 。

图片素材来源于 Instagram
　　* 抱着目的性的摄影虽然能促使自己产出和提升摄影技术，但我觉得这并不是在日常生活中享受摄影最好的方式。抛开这一切之后的世界更令我惊喜和享受。 **

构图

　　这一切都得感谢 Instagram 的出现，所谓无规矩不成方圆，早期的 Instagram 只允许用户上传方形框图的照片，方片摄影。单反的构图比例一般停留在 3 : 2 ，我从最初很难想象一个图片社交平台只允许用户上传方形图到慢慢体验到这种构图给手机摄影带来的“规矩”与美妙再到后来一发不可收拾地爱上这样的构图方式。
　　这样的构图方式很直接的好处就是，多张图的情况下依然能保持统一的切换体验，在朋友圈发图我都会尽量保持这样的风格，也可以算是一种对美的追求了吧。
　　这种构图方式对我影响深远，如果真要说起来都要另开个话题来讲了。

后期

　　后期处理最大的变化在于电脑 PS 转向移动端应用。 vsco 对摄影后期的理解更贴近于我想要的方式。
　　我想要的方式是，只对图片做基本处理加工，而不是像 PS 那样完成抠图，换天空的复杂操作，在手机上完成这些首先太耗时；其次是没必要，对图片的过度加工应该纳入创意摄影的范畴。

如何使用 vsco 完成摄影后期

1. 筛选

　　评估手机图库最近一次拍摄照片，哪些可以进入 vsco 图库

2. 基本处理

　　照片进入 vsco 图库后，对图片进行基本处理。所谓基本处理，主要利用以下三个工具 :

　　旋转工具 : 控制画面平衡的一把手，顾名思义就是调节水平线平衡。 ( 最好在前期拍摄的时候就养成水平线平稳的习惯，有意为之的除外 )
　　拉伸工具 : 控制画面平衡的二把手，用于矫正拍摄角度引起的画面畸变。起因通常是手机与拍摄物不成平行平面导致的。

　　裁剪工具 : 二次 ( 重新 ) 构图好帮手，在使用原生相机拍摄时我会为了尽量保留多的画面而使用默认比例进行拍摄构图 ( 给后期构图留下更多空间 ) ，我一般都会栽成比例 1 : 1 ，二次构图能让画面删繁就简，焕然一新。

　　得益于这三步基本处理，你的照片很可能已经脱胎换骨了。因为这三步之后，照片会趋于工整，工整造就和谐的画面感。

3. 调色

　　选择好一款滤镜，滤镜很多，得益于 vsco 强大高质量的滤镜库，我们可以任意挑选滤镜来满足调色的需要。但是，可能很多人在这里会跟我一开始犯一样的错误 : 滤镜太多 ( 付费后多了几倍 ) 引起了选择困难症，直接导致了我对 vsco 的弃用。我是这样适应的，跟着感觉走，** 选第一印象看中的滤镜，注意做了选择后就不要再思前顾后了，告诉自己已经足够好了。** 千万不要耗费太多的时间在选滤镜上，否则你会对这项工作感到烦躁，进而很可能导致放弃了对该图的处理。一定避免犹豫不决的情况，后续工作就会顺畅许多。
　　我使用最频繁的是彩色滤镜 06 (需付费)，还有几个黑白滤镜。但色彩这全凭个人感觉，因人而异。

4.扩展 : 复制编辑 — 粘贴编辑

　　这个功能可以让你把一张处理好的照片所有的步骤原原本本地应用到另一张照片上面。也就是说，只要把同场景下的其中一张照片处理好了，那么可以直接复制处理步骤到其它几张类似的图上。

谈谈我的摄影观

1.“ Less is more ” ( 摄影就是做减法 )

　　中文相较于英文有一个很大的区别，就是英语两三个句子才能解释得清楚的东西，往往一句中文搞定了。为什么呢，因为中国人很讲究意境。所以中国人玩摄影也很讲究意境。极简主义摄影，仿佛什么都没有，又什么都有。

2.有规有矩，自成方圆

　　规矩在大多数人眼中意味着局限。相反，我觉得遵守“规矩”可以从很大程度上提升手机摄影的质量。
　　我来解析规、矩、方这三个字的含义 : 三种工具，一种构图

  • 规 : 画面平衡 ( 水平线平衡 ) ，对应旋转工具、拉伸工具
  • 矩 : 二次构图，对应裁剪工具
  • 方 : 方形构图

    3.手机摄影，贵在坚持

    　　我分享的并不是什么高水平的修图技巧，于我而言更重要的一点在于如何在日常生活中坚持摄影，而简单明了的修图过程让手机摄影更容易出彩，更容易坚持。
