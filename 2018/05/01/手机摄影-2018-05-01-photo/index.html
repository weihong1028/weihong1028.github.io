生活有许多种记录方式，有的人写下它，有的人讲述它，有的人回忆它。我，想拍下它。

**楼道 / 科技园**
好几次吃饭后路过这个楼道，每次路过都很有意思



**桌面 / 公司**
捣鼓自个的桌面，是件提升幸福感的大事




**优衣库 / 万象城**
万象城店有着不错的外城设计



**公交站台 / 东角头**
清晨的公交站台，新绿的树叶，怡人小景

**点、线、面 / CEEC**
世界可以简单地由点、线、面组成











**仰头 / CEEC**
我喜欢仰天看，拍头上的风景

**Starbucks / 南海意库**
在星巴克呆着远比在家里呆着干活有效率


**少数派 / 公司**
你好，少数派，你会成为自己梦想中的样子吗




**下雨天 / 深大**
雨伞有点漏，不得不停下来抖抖雨水

**路边花 / 科苑**
路边的树开满了小花，路人匆匆，我想稍作停留


**办公新天地 / 公司**
深圳办公室扩租的地盘装修完了，一同来感受感受这满满的科技感






**通宵之战 / 公司**
劳动节前，一场突如其来的黑客马拉松，奋斗的喜悦跃于脸上

再见，四月。
